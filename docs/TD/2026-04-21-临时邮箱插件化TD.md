## 1. 文档目标

本文档解决以下技术实现问题:

1. **注册表机制**: 如何实现 `_REGISTRY` + `register_provider` 装饰器，让 provider 自注册
2. **工厂改造**: 如何将 `get_temp_mail_provider()` 从 if/elif 改为查表
3. **插件目录扫描**: 如何安全地动态 import 第三方 .py 文件
4. **热刷新**: 如何在不重启容器的条件下重新加载插件
5. **插件生命周期 API**: 安装/卸载/配置/测试的完整 API 实现
6. **config_schema**: 插件如何声明配置、前端如何动态渲染表单
7. **CLI 命令**: 如何在 `web_outlook_app.py` 中注册子命令
8. **安全隔离**: 如何防止插件越权和故障扩散
9. **前端集成**: 如何在现有设置页中嵌入插件管理卡片
10. **Docker 适配**: 如何通过 volume 持久化插件文件

**不解决的问题**(留给未来版本):
- 插件沙箱/权限隔离（运行时限制）
- 插件自动热加载（文件变化自动触发）
- 远程 registry 动态拉取
- 插件版本自动更新
- 非 temp_mail 模块的插件化

## 3. 核心技术决策

### 3.1 决策一: 注册表存储位置

**选定方案**: 模块级全局 dict `_REGISTRY`（在 `temp_mail_provider_base.py` 中）

**理由**:
1. Python 模块级 dict 是天然的进程级单例，无需考虑锁（单 worker 部署）
2. 装饰器在模块 import 时执行，注册时机自然
3. 查表是 O(1)，无性能开销

**替代方案**（已否决）:
- `setuptools` entry_points: 过重，用户需 pip install，Docker 内操作复杂
- 数据库表存储: 增加启动时的 DB 依赖，且类对象无法序列化到 SQLite

### 3.2 决策二: 插件加载方式

**选定方案**: `importlib.util.spec_from_file_location()` 动态 import

**理由**:
1. 标准库方案，无额外依赖
2. 可指定文件路径，适合 `plugins/` 目录
3. 加载失败可 try/except 捕获，不影响系统

**替代方案**（已否决）:
- `__import__` + sys.path: 污染全局命名空间，可能与其他模块冲突
- `exec(open(file).read())`: 无法获取类定义，装饰器无法生效

### 3.3 决策三: 热刷新实现方式

**选定方案**: 清空第三方注册 → 重新扫描目录 → 重新 import

**理由**:
1. 实现简单，一个 API 端点 + 一个函数
2. 用户主动触发，可控且安全
3. 内置 provider 不受影响（通过白名单保护）

**不采用**: `watchdog` 文件监控自动触发 — 增加复杂度，且 Docker volume 事件可能不可靠。

### 3.4 决策四: 配置存储格式

**选定方案**: settings 表中 `plugin.{provider_name}.{field_key}` 格式

**理由**:
1. 与现有 settings 读写方式完全一致（`get_setting(key, default)`）
2. 按插件名前缀隔离，不会与内置配置冲突
3. 前端通过 `/api/plugins/{name}/config/schema` 获取 schema，后端读取时拼接 key

## 4. 实现细节：注册表与装饰器

### 4.1 `temp_mail_provider_base.py` 改动

```python
from __future__ import annotations
from abc import ABC, abstractmethod
from typing import Any

# ── 插件注册表 ──
_REGISTRY: dict[str, type[TempMailProviderBase]] = {}

def register_provider(cls: type[TempMailProviderBase]) -> type[TempMailProviderBase]:
    """类装饰器：将 Provider 注册到全局注册表。
    装饰器在模块 import 时执行，注册时机自然。
    """
    name = getattr(cls, "provider_name", "")
    if isinstance(name, str) and name.strip():
        _REGISTRY[name.strip()] = cls
    return cls

def get_registry() -> dict[str, type[TempMailProviderBase]]:
    """返回注册表快照（只读副本）。"""
    return dict(_REGISTRY)


class TempMailProviderBase(ABC):
    provider_name: str = ""
    provider_label: str = ""
    provider_version: str = "0.0.0"
    provider_author: str = ""
    config_schema: dict[str, Any] = {}

    @abstractmethod
    def get_options(self) -> dict[str, Any]: ...

    def generate_mailbox(self, *, prefix: str | None = None, domain: str | None = None) -> dict[str, Any]:
        return self.create_mailbox(prefix=prefix, domain=domain)

    @abstractmethod
    def create_mailbox(self, *, prefix: str | None = None, domain: str | None = None) -> dict[str, Any]: ...

    @abstractmethod
    def delete_mailbox(self, mailbox: dict[str, Any]) -> bool: ...

    @abstractmethod
    def list_messages(self, mailbox: dict[str, Any]) -> list[dict[str, Any]] | None: ...

    @abstractmethod
    def get_message_detail(self, mailbox: dict[str, Any], message_id: str) -> dict[str, Any] | None: ...

    @abstractmethod
    def delete_message(self, mailbox: dict[str, Any], message_id: str) -> bool: ...

    @abstractmethod
    def clear_messages(self, mailbox: dict[str, Any]) -> bool: ...
```

## 5. 实现细节：工厂改造

### 5.1 `temp_mail_provider_factory.py` 完整重写

```python
from __future__ import annotations

import importlib.util
import logging
import sys
from pathlib import Path
from typing import Any

from outlook_web.repositories import settings as settings_repo
from outlook_web.services.temp_mail_provider_base import TempMailProviderBase

logger = logging.getLogger(__name__)

# 内置 provider 白名单（热刷新时不会被卸载）
_BUILTIN_PROVIDERS = {"cloudflare_temp_mail", "custom_domain_temp_mail", "legacy_bridge"}

# ── 确保 import 内置 provider 触发装饰器注册 ──
import outlook_web.services.temp_mail_provider_cf       # noqa: F401, E402
import outlook_web.services.temp_mail_provider_custom    # noqa: F401, E402


class TempMailProviderFactoryError(Exception):
    def __init__(self, code: str, message: str, *, status: int = 503, data: Any = None):
        super().__init__(message)
        self.code = code
        self.message = message
        self.status = status
        self.data = data


def _get_plugin_dir() -> Path:
    """获取插件目录路径。"""
    return Path(__file__).resolve().parents[2] / "plugins" / "temp_mail_providers"


def _get_registry() -> dict[str, type[TempMailProviderBase]]:
    """获取注册表引用。"""
    from outlook_web.services.temp_mail_provider_base import _REGISTRY
    return _REGISTRY


def load_plugins() -> list[dict[str, Any]]:
    """
    扫描 plugins/temp_mail_providers/ 目录，加载第三方插件。
    热刷新时调用此函数重新加载。
    返回: [{"name": str, "status": "loaded"|"failed", "error": str|None}]
    """
    registry = _get_registry()
    plugin_dir = _get_plugin_dir()

    if not plugin_dir.is_dir():
        return []

    results: list[dict[str, Any]] = []
    for py_file in sorted(plugin_dir.glob("*.py")):
        if py_file.name.startswith("_"):
            continue
        name = py_file.stem
        try:
            spec = importlib.util.spec_from_file_location(f"_plugin_{name}", py_file)
            if spec is None or spec.loader is None:
                results.append({"name": name, "status": "failed", "error": "无法创建模块规格"})
                continue
            mod = importlib.util.module_from_spec(spec)
            sys.modules[f"_plugin_{name}"] = mod
            spec.loader.exec_module(mod)
            results.append({"name": name, "status": "loaded"})
            logger.info("[plugin] 已加载: %s", name)
        except Exception as exc:
            results.append({"name": name, "status": "failed", "error": str(exc)})
            logger.error("[plugin] 加载失败 %s: %s", name, exc)

    return results


def reload_plugins() -> dict[str, Any]:
    """
    热刷新：清空第三方插件，重新扫描加载。
    保护内置 provider 不受影响。
    """
    registry = _get_registry()

    # 1. 清空非内置 provider
    removed = [name for name in registry if name not in _BUILTIN_PROVIDERS]
    for name in removed:
        del registry[name]

    # 2. 清理 sys.modules 中的插件模块
    for key in list(sys.modules.keys()):
        if key.startswith("_plugin_"):
            del sys.modules[key]

    # 3. 重新扫描加载
    results = load_plugins()
    loaded = [r["name"] for r in results if r["status"] == "loaded"]
    failed = [r for r in results if r["status"] == "failed"]

    return {
        "loaded": loaded,
        "failed": failed,
        "total_available": len(registry),
    }


def get_temp_mail_provider(provider_name: str | None = None) -> TempMailProviderBase:
    """工厂：根据名称从注册表查找并实例化 provider。"""
    registry = _get_registry()
    resolved = settings_repo.get_temp_mail_runtime_provider_name(provider_name)
    if not resolved:
        raise TempMailProviderFactoryError(
            "TEMP_MAIL_PROVIDER_NOT_CONFIGURED",
            "未配置临时邮箱 Provider",
        )
    cls = registry.get(resolved)
    if cls is not None:
        return cls(provider_name=resolved)
    raise TempMailProviderFactoryError(
        "TEMP_MAIL_PROVIDER_INVALID",
        f"未知的临时邮箱 Provider: {resolved}",
        data={"provider_name": resolved},
    )


def get_available_providers() -> list[dict[str, Any]]:
    """返回所有已注册的 provider 信息。"""
    registry = _get_registry()
    return [
        {
            "name": name,
            "label": getattr(cls, "provider_label", name),
            "version": getattr(cls, "provider_version", "?"),
            "author": getattr(cls, "provider_author", ""),
        }
        for name, cls in sorted(registry.items())
    ]
```

## 7. 实现细节：插件管理服务

### 7.1 新增 `outlook_web/services/temp_mail_plugin_manager.py`

```python
from __future__ import annotations

import hashlib
import json
import logging
import shutil
from pathlib import Path
from typing import Any

logger = logging.getLogger(__name__)

_PLUGIN_DIR = Path(__file__).resolve().parents[2] / "plugins" / "temp_mail_providers"
_REGISTRY_FILE = Path(__file__).resolve().parents[2] / "plugins" / "registry.json"


class PluginManagerError(Exception):
    def __init__(self, code: str, message: str, *, status: int = 400, data: Any = None):
        super().__init__(message)
        self.code = code
        self.message = message
        self.status = status
        self.data = data


def _ensure_plugin_dir() -> Path:
    _PLUGIN_DIR.mkdir(parents=True, exist_ok=True)
    return _PLUGIN_DIR


def get_installed_plugins() -> list[dict[str, Any]]:
    """扫描目录获取已安装插件列表。"""
    _ensure_plugin_dir()
    installed = []
    for py_file in sorted(_PLUGIN_DIR.glob("*.py")):
        if py_file.name.startswith("_"):
            continue
        installed.append({
            "name": py_file.stem,
            "file": py_file.name,
            "size_bytes": py_file.stat().st_size,
        })
    return installed


def get_available_plugins() -> list[dict[str, Any]]:
    """从 registry.json 获取可用插件列表。"""
    if not _REGISTRY_FILE.exists():
        return []
    try:
        data = json.loads(_REGISTRY_FILE.read_text(encoding="utf-8"))
        return data.get("plugins", [])
    except Exception as exc:
        logger.warning("[plugin] registry.json 解析失败: %s", exc)
        return []


def install_plugin(name: str, *, url: str | None = None) -> dict[str, Any]:
    """
    安装插件: 从 registry 或自定义 URL 下载 .py 文件到插件目录。
    """
    _ensure_plugin_dir()
    target = _PLUGIN_DIR / f"{name}.py"

    # 1. 确定下载 URL
    if url:
        download_url = url
    else:
        # 从 registry 查找
        available = get_available_plugins()
        entry = next((p for p in available if p["name"] == name), None)
        if not entry:
            raise PluginManagerError(
                "PLUGIN_NOT_FOUND",
                f"未在官方插件源中找到插件: {name}",
            )
        download_url = entry.get("download_url", "")

    if not download_url:
        raise PluginManagerError("PLUGIN_NO_URL", "未提供下载地址")

    # 2. 下载
    import requests
    try:
        resp = requests.get(download_url, timeout=30, stream=True)
        resp.raise_for_status()
    except requests.Timeout:
        raise PluginManagerError("PLUGIN_DOWNLOAD_TIMEOUT", "插件下载超时")
    except requests.RequestException as exc:
        raise PluginManagerError("PLUGIN_DOWNLOAD_FAILED", f"插件下载失败: {exc}")

    content = resp.content

    # 3. SHA256 校验（可选）
    if url is None:
        entry = next(
            (p for p in get_available_plugins() if p["name"] == name),
            None,
        )
        if entry and entry.get("sha256"):
            actual = hashlib.sha256(content).hexdigest()
            if actual != entry["sha256"]:
                raise PluginManagerError(
                    "PLUGIN_INTEGRITY_CHECK_FAILED",
                    f"文件完整性校验失败: 期望 {entry['sha256'][:16]}...，实际 {actual[:16]}...",
                )

    # 4. 写入
    target.write_bytes(content)
    logger.info("[plugin] 已安装: %s → %s", name, target)

    # 5. 收集依赖信息
    dependencies = []
    if url is None:
        entry = next(
            (p for p in get_available_plugins() if p["name"] == name),
            None,
        )
        if entry:
            dependencies = entry.get("dependencies", [])

    return {
        "plugin_name": name,
        "file_path": str(target),
        "dependencies": dependencies,
    }


def uninstall_plugin(name: str, *, clean_config: bool = False) -> dict[str, Any]:
    """卸载插件: 删除文件，从注册表移除。clean_config=True 时清理关联配置。"""
    target = _PLUGIN_DIR / f"{name}.py"
    if not target.exists():
        raise PluginManagerError("PLUGIN_NOT_INSTALLED", f"插件 {name} 未安装")

    # 检查是否有活跃邮箱
    check = check_provider_in_use(name)
    if check["task_count"] > 0:
        raise PluginManagerError(
            "PLUGIN_IN_USE_BY_TASK",
            f"该插件下有 {check['task_count']} 个进行中的任务邮箱，请先结束任务后再卸载",
            data=check,
        )

    # 删除文件
    target.unlink()

    # 从注册表移除
    from outlook_web.services.temp_mail_provider_base import _REGISTRY
    _REGISTRY.pop(name, None)

    # 清理 sys.modules
    import sys
    sys.modules.pop(f"_plugin_{name}", None)

    # 可选清理关联配置
    cleaned_keys = []
    if clean_config:
        from outlook_web.repositories import settings as settings_repo
        prefix = f"plugin.{name}."
        # 查找并删除所有匹配前缀的 settings
        settings_repo._execute("DELETE FROM settings WHERE key LIKE ?", (f"{prefix}%",))
        cleaned_keys = [f"{prefix}*"]
        logger.info("[plugin] 已清理配置: %s*", name)

    logger.info("[plugin] 已卸载: %s", name)
    return {
        "plugin_name": name,
        "had_active_emails": check["active_count"] > 0,
        "cleaned_keys": cleaned_keys if clean_config else [],
    }


def check_provider_in_use(provider_name: str) -> dict[str, Any]:
    """检查指定 provider 是否有关联的活跃邮箱。"""
    from outlook_web.db import get_db
    db = get_db()
    user_count = db.execute(
        "SELECT COUNT(*) FROM temp_emails WHERE source = ? AND status = 'active'",
        (provider_name,),
    ).fetchone()[0]
    task_count = db.execute(
        "SELECT COUNT(*) FROM temp_emails WHERE source = ? AND status = 'active' AND mailbox_type = 'task'",
        (provider_name,),
    ).fetchone()[0]
    return {
        "in_use": user_count > 0,
        "active_count": user_count,
        "task_count": task_count,
    }


def get_plugin_config_schema(name: str) -> dict[str, Any]:
    """获取插件的配置 Schema。"""
    from outlook_web.services.temp_mail_provider_base import _REGISTRY
    cls = _REGISTRY.get(name)
    if cls is None:
        raise PluginManagerError("PLUGIN_NOT_LOADED", f"插件 {name} 未加载")
    schema = getattr(cls, "config_schema", {})
    return {"plugin_name": name, "config_schema": schema}


def read_plugin_config(name: str) -> dict[str, Any]:
    """读取插件当前配置。"""
    from outlook_web.repositories import settings as settings_repo
    schema = get_plugin_config_schema(name)
    fields = schema.get("config_schema", schema).get("fields", [])
    prefix = f"plugin.{name}."
    config = {}
    for field in fields:
        key = field["key"]
        config[key] = settings_repo.get_setting(f"{prefix}{key}", field.get("default", ""))
    return {"plugin_name": name, "config": config}


def save_plugin_config(name: str, config: dict[str, Any]) -> dict[str, Any]:
    """保存插件配置到 settings 表。"""
    from outlook_web.repositories import settings as settings_repo
    prefix = f"plugin.{name}."
    for key, value in config.items():
        settings_repo.set_setting(f"{prefix}{key}", str(value) if value is not None else "")
    return {"plugin_name": name, "saved_keys": list(config.keys())}


def test_plugin_connection(name: str) -> dict[str, Any]:
    """测试插件连通性。"""
    import time
    from outlook_web.services.temp_mail_provider_factory import get_temp_mail_provider
    start = time.monotonic()
    try:
        provider = get_temp_mail_provider(name)
        options = provider.get_options()
        elapsed_ms = int((time.monotonic() - start) * 1000)
        domains = options.get("domains", [])
        return {
            "success": True,
            "message": "连接成功",
            "latency_ms": elapsed_ms,
            "details": {"domains_count": len(domains)},
        }
    except Exception as exc:
        elapsed_ms = int((time.monotonic() - start) * 1000)
        return {
            "success": False,
            "message": str(exc),
            "latency_ms": elapsed_ms,
        }

    // 保存配置
    async saveConfig(name) {
        const inputs = document.querySelectorAll(`#config-${name} .plugin-config-input`);
        const config = {};
        inputs.forEach(input => {
            const key = input.dataset.key;
            if (!key) return;
            if (input.type === 'checkbox') {
                config[key] = input.checked;
            } else {
                config[key] = input.value;
            }
        });

        const resp = await fetch(`/api/plugins/${name}/config`, {
            method: 'POST',
            headers: {'Content-Type': 'application/json'},
            body: JSON.stringify({config}),
        });
        const result = await resp.json();
        showToast(result.success ? '配置已保存' : (result.message || '保存失败'), result.success ? 'success' : 'error');
    },

    // 测试连通性
    async testConnection(name) {
        const resultDiv = document.getElementById(`test-result-${name}`);
        if (resultDiv) resultDiv.innerHTML = '<div class="test-result" style="border:1px solid var(--border)">测试中...</div>';

        const resp = await fetch(`/api/plugins/${name}/test-connection`, {method: 'POST'});
        const result = await resp.json();

        if (resultDiv) {
            if (result.success) {
                resultDiv.innerHTML = `<div class="test-result test-success">✅ ${escapeHtml(result.message)} · 延迟 ${result.data.latency_ms}ms</div>`;
            } else {
                resultDiv.innerHTML = `<div class="test-result test-fail">❌ ${escapeHtml(result.message)}</div>`;
            }
        }
    },

    // 应用变更（热刷新）
    async applyChanges() {
        const resp = await fetch('/api/system/reload-plugins', {method: 'POST'});
        const result = await resp.json();
        if (result.success) {
            const data = result.data;
            let msg = `插件刷新完成: ${data.loaded.length} 个已加载`;
            if (data.failed.length) {
                msg += `, ${data.failed.length} 个失败`;
            }
            showToast(msg, data.failed.length ? 'warning' : 'success');
        } else {
            showToast('刷新失败', 'error');
        }
    },

    // 自定义安装
    openCustomInstallModal() {
        document.getElementById('modal-custom-install').classList.add('open');
    },
    closeCustomInstallModal() {
        document.getElementById('modal-custom-install').classList.remove('open');
    },
    async customInstall() {
        const name = document.getElementById('custom-install-name').value.trim();
        const url = document.getElementById('custom-install-url').value.trim();
        if (!name || !url) {
            showToast('请填写插件名称和下载地址', 'error');
            return;
        }
        await this.install(name, url);
        this.closeCustomInstallModal();
    },
};
```

## 11. 错误处理与日志

### 11.1 错误码定义

| 错误码 | HTTP | 说明 | 触发场景 |
|--------|------|------|----------|
| `PLUGIN_NOT_FOUND` | 400 | registry 中无此插件 | 安装时插件名不存在 |
| `PLUGIN_NO_URL` | 400 | 无下载地址 | registry 中无 download_url 且未提供自定义 URL |
| `PLUGIN_DOWNLOAD_TIMEOUT` | 504 | 下载超时 | 网络问题 |
| `PLUGIN_DOWNLOAD_FAILED` | 502 | 下载失败 | URL 无效、服务器错误 |
| `PLUGIN_INTEGRITY_CHECK_FAILED` | 400 | SHA256 校验失败 | 文件被篡改 |
| `PLUGIN_NOT_INSTALLED` | 404 | 插件未安装 | 卸载不存在的插件 |
| `PLUGIN_NOT_LOADED` | 404 | 插件未加载 | 读取未加载插件的 schema |
| `PLUGIN_IN_USE_BY_TASK` | 409 | 有进行中的任务邮箱 | 卸载时阻止 |
| `PLUGIN_ALREADY_INSTALLED` | 409 | 插件已安装 | 重复安装 |
| `CONNECTION_FAILED` | 400 | 连通性测试失败 | 配置错误或服务不可用 |
| `INVALID_PARAMS` | 400 | 参数错误 | 缺少 name 或 config 格式错误 |

### 11.2 日志规范

| 操作 | 级别 | 格式 |
|------|------|------|
| 插件加载成功 | INFO | `[plugin] 已加载: {name}` |
| 插件加载失败 | ERROR | `[plugin] 加载失败 {name}: {error}` |
| 插件安装成功 | INFO | `[plugin] 已安装: {name} → {path}` |
| 插件卸载 | INFO | `[plugin] 已卸载: {name}` |
| 热刷新完成 | INFO | `[plugin] 热刷新完成: {loaded} 个加载, {failed} 个失败` |
| 卸载前检查 | WARNING | `[plugin] 插件 {name} 下有 {count} 个活跃邮箱` |

## 15. 风险与缓解

| 风险 | 影响 | 概率 | 缓解措施 |
|------|------|------|----------|
| `importlib` 加载恶意插件 | 安全风险 | 低 | 安装时安全警告；插件约束规则（白名单 import） |
| 热刷新后内存泄漏 | 长时间运行内存增长 | 低 | 清理 `sys.modules`；长时间运行建议定期重启 |
| 插件与项目版本不兼容 | 功能异常 | 中 | `min_app_version` 检查；加载失败优雅降级 |
| 内置 provider 注册被误删 | 系统功能异常 | 极低 | `_BUILTIN_PROVIDERS` 白名单保护；单测覆盖 |
| settings 表 key 冲突 | 配置覆盖 | 极低 | `plugin.` 前缀命名空间隔离 |
| Docker volume 权限问题 | 插件目录不可写 | 低 | 启动时 `mkdir(exist_ok=True)` |
